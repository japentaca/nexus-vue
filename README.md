# Nexus-Vue

Vue.js components for NexusUI audio interfaces.

## Installation

```bash
npm install nexus-vue vue nexusui
```

## Usage

Import the components you need:

```javascript
import { Button, Dial } from 'nexus-vue'
```

Then use them in your Vue app:

```vue
<template>
  <Button />
  <Dial />
</template>
```
